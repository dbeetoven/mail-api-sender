# Node mail sender API

## build with `Node` Api and `React js` in front end

---

## **Plugins**

- express
- body-parser
- dotenv
- cors
- nodemailer
- morgan

---

### **Instalation**

- _node_ `v12` requiere
- `npm install`
- `npm i nodemon -g`

### **Run**

- `nodemon` or `npm run local` or `npm run start`
- .env file requiere.
